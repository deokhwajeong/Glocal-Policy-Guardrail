## How to Update

To replace placeholder images with actual screenshots:

1. Take screenshots of the running application
2. Save them in this directory with appropriate names
3. Update image references in the main README.md:
   ```markdown
   ![Web Dashboard](docs/images/dashboard.png)
   ![Swagger API](docs/images/swagger-ui.png)
   ![Compliance Report](docs/images/compliance-report.png)
   ```

## Image Guidelines

- **Format**: PNG (preferred) or JPG
- **Resolution**: 800x450px minimum (16:9 aspect ratio)
- **Size**: < 500KB per image
- **Quality**: High resolution with clear text visibility
